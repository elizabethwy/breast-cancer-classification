# breast-cancer-classification
# Wisconsin Breast Cancer Classification Analysis

## Overview
This project explores classification modeling techniques using the Wisconsin Diagnostic Breast Cancer dataset in R. The goal of the analysis was to compare the performance of multiple machine learning and statistical classification methods in predicting whether tumors were malignant or benign.

The project focuses on data preprocessing, dimensionality reduction, model training, evaluation, and visualization to better understand patterns within the dataset and assess predictive accuracy.

---

## Objectives
- Analyze diagnostic breast cancer data using statistical and machine learning methods
- Compare the performance of multiple classification models
- Evaluate model accuracy using confusion matrices and performance metrics
- Apply dimensionality reduction techniques to improve interpretation and visualization
- Present findings through visual analysis and formal presentation materials

---

## Methods Used
### Classification Models
- Logistic Regression
- LASSO Regression
- K-Nearest Neighbors (KNN)
- Random Forest

### Dimensionality Reduction
- Principal Component Analysis (PCA)
- Linear Discriminant Analysis (LDA)

---

## Project Workflow
1. Data cleaning and preprocessing
2. Exploratory data analysis
3. Feature selection and scaling
4. Dimensionality reduction using PCA and LDA
5. Classification model implementation
6. Model evaluation and comparison
7. Visualization and interpretation of results

---

## Results
The analysis demonstrated differences in classification performance across models, with ensemble methods providing strong predictive accuracy. PCA and LDA improved visualization and interpretation of feature relationships within the dataset.

The project highlights how statistical learning methods can support medical data analysis and predictive healthcare research.
